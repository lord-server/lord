Display day count = Вывести количество прошедших игровых суток на этом сервере с момента создания
Get help for commands or list privileges = Получить справку по командам или привилегиям

Print server status = Вывести статус сервера
Reload authentication data = Обновить данные аутентификации
Remove IP ban = Удалить IP из бан-листа

Send a private message = Отправить приватное сообщение

Set time of day = Вывести или установить игровое время суток

set volume of sounds, default 1 normal volume. = Установить громкость звука, по умолчанию 1
Show the name of the server owner = Показывать имя создателя сервера
Shutdown server = Выключить сервер
Spawn entity at given (or your) position = Спавн моба на указанной позиции

Teleport to player or position = Переместиться (или переместить другого игрока) на указанную позицию
