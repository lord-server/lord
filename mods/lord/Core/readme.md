`lord/Core/`
===========

Моды общего назначения - общий переиспользуемый ф-ционал, не завязанный на игру.

### `helpers`
 - расширения lua:
   - string: `string.{lower|upper}()` на руском, `string.<...>`
   - table: `.has_key()`, `.contains()`, `merge()`, `overwrite()`, ...
 - доп. хелперы к движку MT
